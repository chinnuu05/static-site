Frequently Asked Questions

What is Lively Wallpaper, is it safe?

Lively Wallpaper is open source software that let's you render videos, images, and HTML as your wallpaper. It's developed by rocksdanister. It is safe and has hundreds of thousands of downloads.

How do I install Notion Gadgets?

Notion Gadgets comes bundled with Lively Wallpaper, a software for rendering HTML as your background. A .exe installer of Lively Wallpaper will be provided after your purchase.

What fields can I add to my database?

Currently only adding the "Title" property of Notion databases is supported.

How do I connect my Notion pages to the widgets?

This is done through Notion itself, in a special template you duplicate. Check out this video for more information.

Why is Lively Wallpaper free but your widgets aren't?

These widgets took a lot of time to make functional, aesthetic, and connect with the Notion API.

Lively Wallpaper is distributed under the GPL v3 license which allows developers to sell their own modified versions of the software, as long as the source code is provided.

Notion Gadgets complies with all GPL v3 licensing regulations as a GitHub fork with the modified version of Lively Wallpaper's source code is public for all to see. The actual HTML, CSS, and JS of the gadgets are not public, but available to customers.

If you are not happy with the $10 price tag, you are free to download the modified version of Lively Wallpaper from GitHub, which enables Notion API support, and create your own CSS/JS/HTML widgets :)
